# AI Scientist WebAPP 用户使用手册

## 1. 简介
AI Scientist WebAPP 是一个用于管理和运行 AI 科学家实验的 Web 平台。通过本系统，您可以方便地管理实验提示词（Prompts）、配置实验参数（YAML）、启动实验并实时查看运行状态及产物。

## 2. 账号注册与登录
本系统采用简化的用户名/密码认证方式。

*   **注册**：点击登录页面的 "Register" 链接，输入用户名和密码即可。
    *   用户名支持：数字、大小写字母、下划线 `_` 和连字符 `-`。
    *   无需邮箱验证。
*   **登录**：使用注册时的用户名和密码登录。
*   **管理员账号**：系统内置管理员账号 `Admin`，密码 `admin123`。

## 3. 提示词管理 (Prompts)
在侧边栏点击 **"Prompts"** 进入提示词管理页面。

### 3.1 创建提示词集 (Prompt Set)
1.  点击页面右上角的 **"Create Prompt Set"** 按钮。
2.  输入提示词集名称（Name）。
3.  点击 **"Create"** 完成创建。

### 3.2 管理版本 (Manage Versions)
1.  在提示词集列表中，点击对应条目的 **"Manage Versions"** 按钮。
2.  在右侧弹出的抽屉中，可以看到该集合下的所有历史版本。
3.  **新建版本**：
    *   点击 **"New Version"**。
    *   输入 **Notes**（版本备注，如 "v2: 优化了设计器角色"）。
    *   在 **Payload (JSON)** 输入框中，以 JSON 格式输入具体的提示词文件内容。
        *   格式示例：`{"system_designer.py": "你的提示词内容...", "other_file.py": "..."}`
    *   点击 **"Create"** 保存。

## 4. 实验运行 (Runs)
在侧边栏点击 **"Runs"** 进入实验列表页面。

### 4.1 创建实验 (Create Run)
1.  点击页面右上角的 **"Create Run"** 按钮。
2.  **Select Prompt Version**：选择要使用的提示词集和版本（可选）。
3.  **YAML Configuration**：配置实验参数。
    *   系统提供默认的 YAML 模板，包含项目名称、最大工作线程数、实验内容描述等。
    *   您可以根据需求直接编辑文本框中的 YAML 内容。
4.  点击 **"Create"** 提交任务。

### 4.2 查看实验详情
在实验列表中点击任意一行，进入实验详情页。

*   **状态监控**：顶部显示当前运行状态（如 `running`, `completed`, `error`）。
*   **操作**：
    *   **Refresh**：手动刷新最新状态。
    *   **Cancel Run**：如果任务正在进行中，可点击取消。
*   **详情标签页**：
    *   **Stages**：以时间轴形式展示实验执行的各个阶段（如 Idea Generation, Code Writing 等）。
    *   **Artifacts**：列出实验生成的产物文件，点击 **"Download"** 即可下载。
    *   **Logs**：查看实验运行的实时日志输出。
